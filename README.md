# Config File Repository
## Bash
- File
    `.bashrc`
- Setup commands
    - Backup current `.bashrc`
        ```sh
        mv .bashrc .bash.bak
    - Download file
        ```sh
        wget https://raw.githubusercontent.com/viscount-monty/config_files/refs/heads/main/.bashrc
    - Source new file
        ```sh
        . ~/.bashrc
